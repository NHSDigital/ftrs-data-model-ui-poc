What is this website for?

This website is using example data to show how data in the new data model will look. We have built this so that developers/users/interested parties may get a feel for the new data model, and how this will affect their work in the future.

What is displayed is NOT a final product and is NOT what will be used by users, this is just for the purposes above. How things are expressed will be more relevant to the roles using the data compared to what is being stored (which is being shown here), and more relevant to how they work with the data.

This website includes the following objects with details about each and relations between them:

  • Healthcare Services
  • Locations
  • Organisations: aligned with ODS not old DoS

You may find terminology that is wrong, or aspects you would want changed, please email: mark.spiers1@nhs.net. We will feed this into our future work to get this service as correct as possible in the future.
